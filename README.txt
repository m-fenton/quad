quad

This program takes height and width values and create a quad as a result. There are five quad options each of which will create a quad with different characters.

To run this program you can go to the main.go file and enter numbers into the function.

In the command line -

go run main.go

For each function there will be a quad created. If zero or a negative number is entered then an error message will print to the command line.

Running the main with the following input will print the below quad to the terminal.

func main () {
QuadA(5,3)
}
